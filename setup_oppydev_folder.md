Run the command in your terminal to download, extract, rename, and clean up the OppyDev project in one go.

```sh
wget https://github.com/OppyDevAI/OppyDev/archive/refs/heads/main.zip -O OppyDev-main.zip && unzip OppyDev-main.zip && mv OppyDev-main oppydev && rm OppyDev-main.zip
```

After running the command, you will have a directory named `oppydev` containing the OppyDev project files.
